# 🌐 LingoBridge

LingoBridge is a modern **language exchange and real-time communication platform** that enables users to connect with people worldwide through instant messaging, voice/video calls, and collaborative features. Built with a scalable architecture, it delivers a seamless communication experience with secure authentication, responsive UI, and powerful real-time capabilities.

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/LingoBridge.git
```

```bash
cd LingoBridge
```

---

## 2️⃣ Backend Environment Variables

Create a `.env` file inside the **backend** folder.

```env
PORT=5001

MONGO_URI=your_mongodb_connection_string

STREAM_API_KEY=your_stream_api_key
STREAM_API_SECRET=your_stream_api_secret

JWT_SECRET_KEY=your_jwt_secret

NODE_ENV=development
```

---

## 3️⃣ Frontend Environment Variables

Create a `.env` file inside the **frontend** folder.

```env
VITE_STREAM_API_KEY=your_stream_api_key
```

---

# 📦 Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

---

# ▶️ Run the Project

### Start Backend

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```

The application will be available at:

```
Frontend: http://localhost:5173
Backend: http://localhost:5001
```
